# Corporate Employee Roster Client (Angular 17)

This is the front-end SPA client built using Angular 17. It interfaces with the Advanced Web API and handles display, query, and modification operations on staff records.

---

## Setup & Execution Instructions

### Prerequisites
- Node.js (v18.0.0 or higher)
- Angular CLI installed globally:
  ```bash
  npm install -g @angular/cli
  ```

### Installation
1. Navigate to the client directory:
   ```bash
   cd Deep_Skilling/Week-5/Angular_Client
   ```
2. Restore package dependencies:
   ```bash
   npm install
   ```

### Execution
1. Boot up the local development server:
   ```bash
   ng serve
   ```
2. Access the dashboard from your browser:
   - URL: `http://localhost:4200`
